# Architecture

## Stack

- Next.js
- Prisma
- PostgreSQL / Supabase
- Vercel

## Direccion tecnica

- `core` obligatorio para capacidades base.
- `modules` opcionales por tenant.
- `organizationId` como eje multi-tenant.
- Controles server-side obligatorios.
- Activacion/desactivacion de modulos sin forks.
