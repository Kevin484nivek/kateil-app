# FAQ operativa MiMarca

## ¿Cómo hago una venta rápida?

Usa `Nueva venta`, añade producto, cantidad y confirma. Si necesitas precio final exacto, usa `Precio final deseado` para que el sistema calcule descuento equivalente.

## ¿Cómo hago una devolución con cambio?

En `Nueva venta`, activa `Devolución / cambio`, selecciona ticket original, añade líneas devueltas y añade las nuevas líneas que se lleva la clienta.

## ¿Cómo cargo mucha mercancía de una vez?

En `Mercancía`, selecciona proveedor y añade varias líneas en el mismo registro. El sistema permite crear productos en ese mismo flujo para acelerar altas masivas.

## ¿Cómo reviso el estado de un producto?

En `Productos` puedes filtrar y buscar por código, nombre, categoría o proveedor. También puedes revisar trazabilidad en `Movimientos`.

## ¿Dónde se guardan contratos y adjuntos?

Con integración de Drive conectada, los adjuntos de proveedores y mercancía se suben a las carpetas preparadas automáticamente.

## ¿Qué pasa si falla un backup?

El sistema registra el error en logs y envía notificación por Telegram al canal de estado operativo.
