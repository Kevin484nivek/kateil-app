# Manual de usuario MiMarca

## Objetivo

MiMarca centraliza ventas, stock, mercancía, proveedores, clientes, finanzas y usuarios internos.

## Nueva venta

1. Entra en `Nueva venta`.
2. Busca productos por código, nombre, proveedor o categoría.
3. Ajusta cantidad, precio y descuentos por línea o global.
4. Selecciona forma de pago.
5. Revisa el resumen y confirma.

### Devolución o cambio

1. Cambia a modo `Devolución / cambio`.
2. Selecciona el ticket original.
3. Añade líneas devueltas desde ese ticket.
4. Si la clienta se lleva otro producto, añádelo en el mismo ticket.
5. El sistema calcula el saldo neto final.

## Mercancía

1. Selecciona proveedor.
2. Añade líneas de entrada con producto, cantidad y coste.
3. Puedes crear producto en el mismo flujo.
4. Al guardar:
   - sube adjuntos al almacenamiento configurado
   - actualiza stock
   - registra movimientos
   - genera gasto de mercancía en finanzas

## Proveedores

- Gestiona datos de contacto y contrato PDF.
- Configura tipo de liquidación.
- Los adjuntos se sincronizan con Drive cuando la integración está conectada.

## Finanzas

- Control mensual de gastos e ingresos.
- Incluye gastos manuales, recurrentes y los generados por mercancía.

## Usuarios y almacenamiento

- `Usuarios` permite revisar estado, rol y última conexión.
- `Usuarios > Almacenamiento` permite conectar y validar Google Drive.

## Backups y restauración

- Backup diario en servidor.
- Copia del backup en Google Drive.
- Prueba semanal de restauración segura sin solape con backup diario.
