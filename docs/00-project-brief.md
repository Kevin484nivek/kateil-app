# Project Brief

## Objetivo

Construir la base tecnica de Kateil como plataforma SaaS modular multi-tenant para retail fisico, evolucionando desde la experiencia y bloques funcionales de MiMarca.

## Alcance actual

- Baseline Next.js + Prisma + PostgreSQL.
- Enfoque Vercel + Supabase.
- Core multi-tenant con organizaciones, membresias y modulos.
- Documentacion de producto, arquitectura, operacion y readiness en `docs/kateil/`.

## Principios

- No reescribir desde cero.
- Extraer progresivamente bloques desde la base MiMarca.
- Evitar forks por cliente.
- Clasificar capacidades como `core` o `module`.
